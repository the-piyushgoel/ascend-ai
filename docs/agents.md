# Agents Documentation

## Resume Agent

Purpose:

Analyze user resumes.

Input:

* Resume PDF

Output:

* Skills
* Projects
* Experience
* Achievements
* ATS Score
* Resume Suggestions

---

## GitHub Agent

Purpose:

Analyze coding profile and development activity.

Input:

* GitHub Username

Output:

* Repository Count
* Commit Activity
* Languages Used
* Stars
* Activity Score

---

## Assessment Agent

Purpose:

Evaluate technical readiness.

Input:

* MCQs
* Fill Ups
* Short Text Answers

Output:

* Topic Wise Scores
* Overall Assessment Score
* Strengths
* Weaknesses

---

## Gap Analysis Agent

Purpose:

Identify missing requirements for the target role.

Input:

* Resume Analysis
* GitHub Analysis
* Assessment Results
* Career Goal

Output:

* Missing Skills
* Missing Projects
* Missing Experience
* Improvement Suggestions

---

## Roadmap Agent

Purpose:

Generate personalized roadmaps.

Input:

* Career Goal
* Timeline
* Daily Study Hours
* Gap Analysis

Output:

* Must Do Tasks
* Recommended Tasks
* Weekly Milestones
* Estimated Timeline

---

## Mentor Agent

Purpose:

Provide interactive career guidance.

Input:

* User Questions

Output:

* Personalized Guidance
* Roadmap Updates
* Career Advice

---

## Orchestrator Agent

Purpose:

Coordinate all agents and manage workflow.

Responsibilities:

* Trigger specialized agents
* Aggregate outputs
* Calculate readiness score
* Generate final recommendations
* Manage overall user journey
