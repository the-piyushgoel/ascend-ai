# Ascend AI Architecture

## High Level Flow

User
↓
Goal Setup
↓
Resume Agent
↓
GitHub Agent
↓
Assessment Agent
↓
Gap Analysis Agent
↓
Roadmap Agent
↓
Mentor Agent

---

## Agents

### Resume Agent
Extracts:
- Skills
- Experience
- Projects
- Achievements
- Contact Information
- ATS Insights

### GitHub Agent
Analyzes:
- Repositories
- Stars
- Languages
- Contributions
- Commit Activity

### Assessment Agent
Evaluates:
- DSA
- OOP
- OS
- DBMS
- CN

### Gap Analysis Agent
Identifies:
- Missing Skills
- Missing Projects
- Missing Experience

### Roadmap Agent
Generates:
- 30 Day Plan
- 60 Day Plan
- 90 Day Plan

### Mentor Agent
Provides:
- Personalized Recommendations
- Dynamic Roadmap Updates
